# RougeUI

This is my **_personal_** UI for Wotlk-Classic
# **Feature List:**

To open the addon's settings navigate in Interface Options > AddOns or write /rui

See https://www.curseforge.com/wow/addons/rougeui for a list of features

# **Read Me:**

All options listed above can be enabled **MANUALLY**. After enabling one or multiple options you will have to reload.
